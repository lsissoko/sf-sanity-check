### Salesforce Config
create a `data/config.json` file with your Salesforce credentials:
```
{
  "salesforce": {
    "version": "",
    "clientId": "",
    "clientSecret": "",
    "clientPassword": "",
    "clientUserName": "",
    "salesforceUrl": ""
  }
}
```
